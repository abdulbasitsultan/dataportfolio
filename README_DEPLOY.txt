HOW TO DEPLOY THIS PORTFOLIO ON NETLIFY - NO CODING NEEDED

1. Unzip the folder.
2. Open Netlify.
3. Go to Projects.
4. Under "Upload your project files", drag and drop the whole unzipped folder called abdul_portfolio_site.
5. Wait for Netlify to publish it.
6. Netlify will give you a free link ending in .netlify.app.

WHAT TO EDIT LATER
- index.html contains all the website text.
- style.css controls the design.
- assets/images contains the portfolio images.
- assets/downloads contains downloadable PDFs.

IMPORTANT
Do not upload only index.html. Upload the full folder so the images and downloadable PDFs work.
